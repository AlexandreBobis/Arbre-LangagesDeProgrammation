# Énoncé de mini projet à créer en une journée pour ancrer la notion Projet en JS

Pour commencer : Créer une page HTML. On a une liste d’ingrédients pour la recette carbonara.

1. Pâtes
2. Oeuf
3. Sel
4. Poivre
5. Lardon
6. Oignon
7. Parmesan

<span style="color:green">Première étape :</span> La liste des ingrédients doit être stockée dans un tableau. Afficher en une phrase, le nombre d’ingrédients nécessaires pour la recette.

<span style="color:green">Deuxième étape :</span> Afficher la liste des ingrédients nécessaires pour la recette des pâtes carbonara. Pas de mode d’affichage imposé.

<span style="color:green">Troisième étape :</span> L’affichage doit se faire avec un numéro attribué à l’ordre, comme dans la liste ci-dessus. On doit revenir à la ligne pour chaque ingrédient.

<span style="color:green">Quatrième étape :</span> L’ordre d’affichage doit se faire de manière alphabétique.
